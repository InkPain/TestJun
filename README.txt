# Тестовое задание на должность Junior Unity developer
! В папке Scripts/Gameplay находятся скрипты отвечающие за гейплей

! В папке Scripts/UI находятся скрипты отвечающие за интерфейс и работу с ним

!Реализация движения корабля скриптами в папке ShipControllers

Для движения карабля используйте клавиши A и D 
Для стрельбы простым оружием используйте клавишу SPACE
Для стрельбы ракетой используйте клавишу R

Есть возможность движения мышью для переключения клавиша Q
Для движения переместите мышь в нужном направлении
Для стрельбы используйте ЛКМ
Для стрельбы ракетами используйте ПКМ

!В папке Gameplay/Gamesystem находятся скрипты отвечающие за:
То как и когда происходит поражение
За вычет ХП
За телепорт если игрок выходит за зону

!В папке Gameplay/Bufs находятся скрипты отвечающие за систему бафов
Выподение бафов происходит когда умирает вражеский корабль, это реализовано в скрипте кораблей (Gamplay/Spacesships/Spaceship)

!За показ очков отвечают скрипты в папке Scritps/UI/Score
начисление очков происходит при сбитии корабля. это реализовано в скрипте кораблей (Gamplay/Spacesships/Spaceship)

!Система отображения хп реализована скриптами в папке Scripts/UI/HPSystem
